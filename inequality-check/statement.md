## Problem Statement 

Given two integers $a$ and $b$, check if $a < b$ holds.

## Constraints

$0 \leq a, b \leq 100$

## Input

The input contains two integers $a$ and $b$ in one line.

## Output

Output "Yes" if the inequality holds. Otherwise, output "No".

## Examples

### Input

```
3 5
```

### Output

```
Yes
```

### Input

```
77 77
```

### Output

```
No
```
