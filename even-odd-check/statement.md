## Problem Statement 

Given an integer $n$, check if $n$ is even or odd.

## Constraints

$-100 \leq n \leq 100$.

## Input

The input contains one integer, $n$.

## Output

If $n$ is even, output "even". Otherwise, output "odd".

## Examples

### Input

```
-41
```

### Output

```
odd
```

### Input

```
100
```

### Output

```
even
```
